# 🐧 Debian 13 VM Chat

[![Website](https://img.shields.io/badge/Инструкция-Онлайн-blue?style=flat&logo=github)](https://maxgamers16.github.io/)

Этот репозиторий содержит простой, но функциональный чат на `Python 3`, созданный для общения между двумя и более виртуальными машинами под управлением **Debian 13** в одной локальной сети.

## 🚀 Быстрый старт

1. Скачай архив `python-code.zip` со страницы [Релизов](https://github.com/MaxGamers16/MaxGamers16.github.io/releases/latest).
2. Распакуй его: `unzip python-code.zip`.
3. На одной машине запусти сервер: `python3 server.py`.
4. На другой машине запусти клиент: `python3 client.py` и введи IP-адрес сервера.

## 📚 Полная инструкция

Все детали по настройке сети в VirtualBox, работе с файрволом (`ufw`) и устранению неполадок доступны на веб-странице проекта.
