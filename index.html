# TALK IN TERMINAL

DEBIAN
CHAT 1.0

Простой TCP‑чат на Python для виртуальных машин.
Общайтесь между VM в локальной сети.
⬇ СКАЧАТЬ ZIP GITHUB REPO
⚡ ВАЖНО: Виртуальные машины должны быть в одной сети.
В VirtualBox используйте «Сетевой мост» или внутреннюю сеть.
// ИНСТРУКЦИЯ

01. УСТАНОВКА

Убедитесь, что Python 3 установлен:

sudo apt update && sudo apt install python3 -y

Скачайте архив и распакуйте:

unzip python-code.zip

02. СЕРВЕР

На первой VM (сервер) выполните:

python3 server.py

Появится сообщение о запуске на порту 12345.

03. КЛИЕНТ

На второй VM запустите клиент:

python3 client.py

Введите IP сервера (узнать командой ip a).

04. ОБЩЕНИЕ

Всё готово! Набирайте сообщения и жмите Enter.

Для выхода нажмите Ctrl+C.

// НАСТРОЙКА СЕТИ В VIRTUALBOX

🌐 СЕТЕВОЙ МОСТ

Чтобы виртуальные машины видели друг друга и получали IP из одной подсети с хостом:

  • Откройте настройки VM → Сеть.
  • Тип подключения: Сетевой мост.
  • Выберите ваш физический сетевой адаптер (обычно eth0 или Wi‑Fi).

✅ IP-адрес будет выглядеть примерно как 192.168.1.5 или 192.168.0.10.

📡 КАК УЗНАТЬ IP

На каждой виртуальной машине выполните:

ip a | grep inet

Ищите адрес, начинающийся с 192.168. или 10.0.. Это и есть IP в локальной сети.

Запомните: IP может меняться после перезагрузки, если не настроен статический адрес.

  • ❓ IP отличается от примера? Это нормально! Главное, чтобы оба IP были из одной подсети (например, 192.168.1.X). Вводите именно тот IP, который показала команда ip a на сервере.
  • 🚫 Другой человек не может подключиться?
    • Убедитесь, что обе VM действительно в одной локальной сети (LAN). При использовании NAT они изолированы.
    • Проверьте, не блокирует ли файрвол порт 12345 на сервере:
      sudo ufw allow 12345/tcp
    • Попробуйте пропинговать сервер с клиента: ping 192.168.X.X. Если пинг не проходит — проблема на уровне сети VirtualBox.
  • 🔄 Всё равно не работает? Переключите обе VM на «Внутренняя сеть» (Internal Network) с одинаковым именем. Тогда они будут видеть только друг друга, но не получат доступ в интернет.
