💬 Чат для виртуальных машин Debian 13

Этот репозиторий содержит простой TCP-чат на Python для общения между виртуальными машинами в локальной сети.

⬇ Python-code.zip

⚠️ Важно: Для работы чата обе виртуальные машины должны находиться в одной сети (например, в VirtualBox настроен "Сетевой мост" или внутренняя сеть).

🚀 Инструкция по запуску в Debian 13

  1. Убедитесь, что установлен Python 3:
    sudo apt update && sudo apt install python3 -y
  2. Скачайте файлы на обе виртуальные машины (через браузер, curl или общую папку).
  3. На сервере (VM1):
    python3 server.py
    Вы увидите сообщение "Сервер запущен на порту 12345...".
  4. На клиенте (VM2):
    python3 client.py
    Введите IP-адрес сервера (узнать его можно командой ip a на сервере) и начинайте общаться.

🛠 Примечания

  • Порт 12345 можно изменить в коде (переменная PORT).
  • Если возникают проблемы с подключением, проверьте, что фаервол не блокирует порт:
    sudo ufw allow 12345/tcp
