💬 Debian Chat v1.0 · TCP

Простой чат на Python для виртуальных машин Debian 13. Общайтесь между VM в локальной сети быстро и без лишнего.
⬇ Скачать python-code.zip GitHub Repo

📦 Архив содержит server.py и client.py.

🚀 Быстрый старт в Debian 13

⚠️ Убедитесь, что виртуальные машины находятся в одной сети (VirtualBox: «Сетевой мост» или внутренняя сеть).
  1. Установите Python 3 (если ещё нет):
    sudo apt update && sudo apt install python3 -y
  2. Скачайте архив на обе виртуальные машины и распакуйте:
    unzip python-code.zip
  3. На сервере (VM1):
    python3 server.py
    Появится сообщение «Сервер запущен на порту 12345...».
  4. На клиенте (VM2):
    python3 client.py
    Введите IP-адрес сервера (узнать можно командой ip a).

🛠️ Полезные заметки

  • Порт по умолчанию 12345. Изменить можно в переменной PORT внутри скриптов.
  • Если подключение не устанавливается, разрешите порт в файрволе:
    sudo ufw allow 12345/tcp
  • Для выхода из чата нажмите Ctrl+C.
